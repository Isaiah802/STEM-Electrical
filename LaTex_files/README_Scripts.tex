\section{TROUBLESHOOTING}

===============================================================================\par

Problem: "pip: command not found"\par

Solution: Python not installed or not in PATH.\par

\begin{itemize}
  \item 1. Verify Python installed: python --version
  \item 2. If not installed, download from python.org
  \item 3. During installation, CHECK "Add Python to PATH"
\end{itemize}

Problem: "python-pptx not found" or "reportlab not found"\par

Solution: Libraries not installed.\par

\begin{itemize}
  \item Run: pip install python-pptx reportlab
\end{itemize}

Problem: Script runs but no output\par

Solution: Check for error messages in console.\par

\begin{itemize}
  \item For PowerPoint: Verify no existing file is open (close PowerPoint)
  \item For PDFs: Check "PDFs/" folder was created
\end{itemize}

Problem: PDF formatting looks weird\par

Solution: Text files may have unusual characters.\par

\begin{itemize}
  \item The script handles most ASCII art automatically
  \item If issues persist, check text file encoding (should be UTF-8)
\end{itemize}

Problem: "Permission denied" error\par

Solution: File is open in another program.\par

\begin{itemize}
  \item Close PowerPoint before re-running create\_presentation.py
  \item Close PDF viewer before re-running create\_pdfs.py
\end{itemize}

\section{BEFORE THE MEETING:}

\begin{itemize}
  \item 1. Run both scripts to generate PowerPoint and PDFs
  \item 2. Review the PowerPoint presentation (18 slides, \textasciitilde{}3-4 min per slide)
  \item 3. Print PDFs for distribution:
  \item 1 copy per team member + 2 extras
  \item \textasciitilde{}90 pages total per set
  \item Consider: Print Safety\_Protocol.pdf separately (signature required)
  \item 4. Test PowerPoint on meeting room equipment
\end{itemize}

DURING THE MEETING (1 hour):\par

\begin{itemize}
  \item 0-5 min: Introduction (Slides 1-3)
  \item 5-20 min: Architecture \& Power Budget (Slides 4-7)
  \item 20-30 min: Safety Protocols (Slide 8) - Mandatory
  \item 30-40 min: Subsystem Assignments (Slides 9-13)
  \item 40-50 min: Action Plan (Slides 14-15)
  \item 50-55 min: Resources \& Q\&A (Slides 16-17)
  \item 55-60 min: Closing \& Next Steps (Slide 18)
\end{itemize}

\begin{itemize}
