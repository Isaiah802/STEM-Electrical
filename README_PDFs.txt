TO REGENERATE PDFs:
─────────────────────────────────────────────────────────────────────────

If you need to modify and recompile the documents:

1. Edit the source file in LaTex_files/
2. Run: python compile_latex.py
3. PDFs will automatically be moved to PDFs folder

The script now AUTO-MOVES compiled PDFs to the PDFs folder!


TO CREATE NEW DOCUMENTATION:
─────────────────────────────────────────────────────────────────────────

1. Create a .txt file in txt_files/
2. Add it to create_latex_documents.py conversions list
3. Run: python create_latex_documents.py
4. Run: python compile_latex.py
5. Your PDF will appear in PDFs folder
