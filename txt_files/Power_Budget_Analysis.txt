CRITICAL ANALYSIS:
─────────────────────────────────────────────────────────────────────────
⚠ WARNING: ESC BEC (3A) is INSUFFICIENT for worst-case scenarios

The 5V 3A BEC provides 3000mA maximum. Under aggressive maneuvering or
servo stall conditions, total system draw exceeds BEC capacity.

FAILURE MODES:
  1. BEC voltage sag → Flight controller brownout → Loss of control
  2. BEC thermal shutdown → Complete power loss → Crash
  3. ESC protective cutoff → Motor stops mid-flight → Crash

SECTION 5: RECOMMENDED POWER ARCHITECTURE
═══════════════════════════════════════════════════════════════════════════

OPTION A: Single BEC Configuration (NOT RECOMMENDED)
─────────────────────────────────────────────────────────────────────────
  ESC BEC (5V 3A) → ALL systems
  
  Pros:  Simplicity, minimal wiring
  Cons:  ✗ Insufficient current for servo stall conditions
         ✗ Single point of failure
         ✗ Risk of brownout → flight controller reset → crash
         ✗ No engineering margin

  VERDICT: UNSAFE for portfolio-level aerospace work


OPTION B: Dual-BEC Configuration (RECOMMENDED) ✓
─────────────────────────────────────────────────────────────────────────
  Power Rail 1 (ESC BEC):        Power Rail 2 (External BEC):
    - Flight Controller            - 4× Servos
    - GPS Module                   (High current isolation)
    - ELRS Receiver
    
  Power Rail 3 (Arduino Nano onboard regulator from 5V):
    - BME280 Sensor
    - MQ135 Sensor
    - MicroSD Module

  Pros:  ✓ Isolated servo power prevents brownout
         ✓ Redundancy (if servo BEC fails, FC stays alive)
         ✓ Engineering margin: ~1.5A on FC rail
         ✓ Professional-grade power distribution
         ✓ Demonstrates systems engineering competency

  VERDICT: CORRECT ENGINEERING CHOICE for serious aerospace project


OPTION C: Hybrid Configuration (ACCEPTABLE ALTERNATIVE)
─────────────────────────────────────────────────────────────────────────
  ESC BEC → FC + GPS + Receiver + Nano/Sensors
  External BEC → Servos ONLY
  
  Advantage: Maximum servo current headroom
  Use if: You want absolute power isolation for control surfaces

SECTION 6: POWER DISTRIBUTION IMPLEMENTATION
═══════════════════════════════════════════════════════════════════════════

Required Additional Hardware:
  ☐ 5V UBEC (5-6A rating recommended)
     Example: Hobbywing 5V 6A UBEC or similar
            Input: 7-26V (connect to main LiPo via XT60 breakout)
            Output: 5V @ 6A (more than enough for 4 servos)
  
  ☐ XT60 Y-Harness or Distribution Board
     To split main battery power to ESC + External BEC

  ☐ Power Distribution PCB (optional but recommended)
     Clean solder joints for all 5V connections
     Common ground plane

Wiring Strategy:
  1. Main LiPo → XT60 Splitter
     ├─→ ESC (motor + 5V BEC output)
     └─→ External BEC (5V output to servo rail)
  
  2. ESC BEC 5V Output → FC + GPS + Receiver rail
  
  3. External BEC 5V Output → Servo power pins
  
  4. Nano: Powered from FC 5V rail or external BEC
  
  5. ALL GROUNDS MUST BE COMMON
     Critical: Connect all ground wires together

SECTION 7: RISK ASSESSMENT & MITIGATION
═══════════════════════════════════════════════════════════════════════════

Risk #1: BEC Overload During Flight
  Likelihood:   HIGH (if using single BEC)
  Impact:       CATASTROPHIC (loss of aircraft)
  Mitigation:   Use dual-BEC architecture (Option B)
  Validation:   Bench test with all servos under load

Risk #2: Electrical Noise from ESC/Motor
  Likelihood:   MEDIUM
  Impact:       MODERATE (sensor errors, GPS dropouts)
  Mitigation:   - Twist motor wires
                - Route motor wires away from signal wires
                - Use ferrite beads on ESC power lines
                - Keep MQ135 sensor away from high-current paths
  Validation:   Monitor GPS lock during motor operation on bench

Risk #3: Ground Loop / Poor Grounding
  Likelihood:   MEDIUM
  Impact:       MODERATE (erratic behavior, noise)
  Mitigation:   - Star ground topology from battery negative
                - Use thick ground wires (18-20 AWG)
                - Verify continuity across all grounds
  Validation:   Multimeter continuity test

Risk #4: Servo Stall Current Spike
  Likelihood:   LOW (good airframe should prevent jamming)
  Impact:       HIGH (voltage sag, brownout)
  Mitigation:   - Dual-BEC architecture
                - Ensure control surfaces move freely
                - Add current limiting if needed
  Validation:   Manual obstruction test on bench (brief)

Risk #5: LiPo Mishandling
  Likelihood:   MEDIUM (team inexperience)
  Impact:       CATASTROPHIC (fire, injury)
  Mitigation:   - Mandatory safety training
                - LiPo safety bag usage
                - Low-voltage alarm
                - Never charge unattended
  Validation:   Supervised charging protocol

Risk #6: MQ135 Sensor Warm-Up
  Likelihood:   CERTAIN (MQ135 requires 24-48hr burn-in)
  Impact:       LOW (inaccurate gas readings initially)
  Mitigation:   - Power sensor for 24-48 hours before first flight
                - Document calibration procedure
                - Log warm-up period in data
  Validation:   Monitor sensor readings over 48-hour period

SECTION 8: VALIDATION & TESTING REQUIREMENTS
═══════════════════════════════════════════════════════════════════════════

PHASE 1: Component-Level Testing
  ☐ Verify all component voltage ratings (multimeter)
  ☐ Test each servo individually (movement, current draw)
  ☐ Confirm GPS lock indoors/outdoors
  ☐ Validate Arduino sensor readings
  ☐ Measure ESC BEC output under no load: Expected 5.0V ± 0.1V

PHASE 2: Subsystem Integration Testing
  ☐ Power test bench: ESC + BEC + FC + GPS + Receiver
  ☐ Measure current draw at idle: Expected ~320mA
  ☐ Motor spin test (NO PROP): Verify ESC signal response
  ☐ Servo bench test: All 4 servos moving simultaneously
     → Measure current with clamp meter: Should NOT exceed 2000mA
  ☐ Arduino data logging: Confirm MicroSD writes correctly

PHASE 3: Full System Integration
  ☐ All systems powered simultaneously
  ☐ Measure voltage at FC under servo load: Must stay > 4.8V
  ☐ GPS lock test with motor running: Verify no GPS dropouts
  ☐ Range test: Verify receiver connection at 100m
  ☐ Vibration test: Secure all connections

PHASE 4: Ground Validation
  ☐ Control surface checks: Full deflection, no binding
  ☐ Failsafe test: Turn off transmitter → verify servo behavior
  ☐ Battery alarm test: Verify low voltage warning at 14.0V
  ☐ Emergency shutdown procedure: Kill switch test
